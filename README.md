# ⚙️ 자동화설비기능사 실기 · PLC제어작업 마스터

자동화설비기능사 **실기(PLC 제어작업)** 대비 학습 웹앱입니다. 제어 개념을 단계별로 익히고,
PLC 동작을 시운전 시뮬레이터로 체험한 뒤, 테스트로 확인합니다. 설치 없이 브라우저에서 바로 실행됩니다.

🔗 **바로가기:** https://hongyul67-cpu.github.io/automation-plc-exam/

## 주요 기능
- **단계별 학습** — 제어 개념을 이전/다음으로 넘기며 학습
- **시운전 시뮬레이터** — PB(푸시버튼) 연속동작 · 비상정지 · 공작물 재적재 등 실제 동작 흐름 체험
- **테스트** — 입출력(IO) 배치 · 래더 · 회로 문제를 풀고 즉시 채점

## 사용 방법
- 위 주소를 열면 바로 사용할 수 있습니다. (PC·태블릿·스마트폰 지원, 인터넷 연결만 필요)

## 구성
- `index.html` — 앱 본체
- `data.js` — 학습·문제 데이터
